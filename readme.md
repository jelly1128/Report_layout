# Report Layout
リアルタイム解析システムの「レポート生成・印刷機能」を担当するPythonサブシステムの開発・デバッグ用プロジェクト

C++側が出力した解析結果（JSON）を読み込み，HTMLテンプレートを用いて検査要約レポート（PDF）を生成・印刷する．

## 本プロジェクトの位置づけと使用用途
### 位置づけ
1. **C++解析エンジン:** 解析実行 -> 結果を `report.json` として出力
2. **本プロジェクト (Python):** `report.json` を読み込み -> HTMLに埋め込み -> PDF生成 -> 印刷

### 使用用途
* C++側の処理を回さずに，レポート生成部分単体での動作確認．
* HTML/CSSを変更してのレイアウト修正・調整．
* JSONデータを手動で書き換え，様々な解析結果パターンにおける表示崩れのチェック．

## 構成概要
* **入力:** `report.json` (C++の出力形式に準拠)
* **レイアウト:** 
`report_jpn.html` (日本語テンプレート), `report_eng.html` (英語テンプレート)
* **処理ロジック:** `print_report.py`
* **出力:** PDFファイル生成 および プリンターへの送信
